Robot Framework

Robot Framework é uma ferramenta de automação genérica e de código aberto para testes de aceitação, desenvolvimento orientado por testes de aceitação (ATDD) e automação de processos robóticos (RPA).

Características Principais

  • Sintaxe simples: Tabular com keywords em linguagem natural
  • Bibliotecas extensíveis: Requests, SeleniumLibrary, DatabaseLibrary
  • Relatórios ricos: HTML reports com logs detalhados
  • Integração CI/CD: Fácil integração com GitHub Actions
  • Multiplataforma: Python, Java, .NET

Projeto Atual: Utilizado para automação de testes de API REST com validações completas de CRUD, cenários negativos e testes de regressão.
